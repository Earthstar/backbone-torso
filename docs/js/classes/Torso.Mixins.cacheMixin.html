cacheMixin.registerIds

(
  • newIds
  • guid
)

Defined in /other/dev/backbone-torso/modules/mixins/cacheMixin.js:258

Registers a list of Ids that a particular object cares about and pushes any cached models its way.

This method intelligently updates the "_requestedIds" field to contain all unique requests for Ids to be fetched. Furthermore, the "polledFetch" method is overriden such that it no longer routes through Backbone's fetch all, but rather a custom "fetchByIds" method.

Parameters:

  • newIds Array
    • New ids to register under the requester
  • guid String
    • The GUID of the object that wants the ids

constructor

)

Defined in /other/dev/backbone-torso/modules/mixins/cacheMixin.js:413

The constructor constructor / initialize method for collections. Allocate new memory for the local references if they were null when this method was called.

Parameters:

  • [options] Object optional
    • optional options object
    • [fetchHttpAction='POST'] String optional

      http action used to get objects by ids

    • [getByIdsUrl='/ids'] String optional

      path appended to collection.url to get objects by a list of ids

    • [lazyFetch=false] Boolean optional

      if set to true, the collection will assume that models do not change after first fetch

    • [fetchUsingTrackedIds=true] Boolean optional

      if set to false, cache.fetch() will not pass to fetchByIds with current tracked ids but will rather call the default fetch method.
